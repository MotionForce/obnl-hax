<!-- GETTING STARTED -->
## Getting Started


### Prerequisites
Install Node.js
* [node.js](https://nodejs.org/en/)

Install npm
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MotionForce/OBNL-HAX.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
